# GARGANTUA.exe

A real-time black hole renderer built in Three.js and GLSL because 
I watched Project Hail Mary at 11pm and made questionable decisions.

# AMAZE AMAZE AMAZE - Rocky

## What is this

A raymarched black hole visualiser implementing Schwarzschild and Kerr 
geodesics with post-processing effects. It runs in your browser. 

## What it does

- Real-time raymarching of a black hole in WebGL
- Schwarzschild and Kerr metric approximations
- Gravitational lensing, accretion disc, and photon sphere
- Bloom, chromatic aberration, and other post-processing effects
- Runs at a frame rate that will surprise you

## Stack

Three.js, custom GLSL shaders, WebGL, and an unreasonable 
amount of astrophysics Wikipedia tabs.

## Live

https://blackhole-visualiser.vercel.app/

## Usage

Clone it. Run it. Stare into the void.

```bash
git clone https://github.com/san5kriti/gargantua
npm install
npm run dev
```
